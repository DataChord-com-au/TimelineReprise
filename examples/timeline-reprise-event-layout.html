SIMILE Timeline Reprise: Event Layout

This example shows event layout changes on horizontal and vertical timelines.

Horizontal Timeline

  • duration labels are routed away from crowded event tracks
  • short duration events remain visible without taking over the layout
  • sticky range labels slide at both viewport edges while their tapes remain visible
  • the overview band uses theme settings from the same timeline

Vertical Timeline

  • duration and point events use the same data as the horizontal timeline
  • labels are orthogonal to the time direction, so layout pressure moves sideways
  • the overview band sits beside the main band instead of below it
